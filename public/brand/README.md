# QuantXlr8 Brand Assets

## Brand Overview
**QuantXlr8** (short name: **QX8**) is a venture-client accelerator connecting startups with enterprise business units for real pilots that convert to paid deployments.

## Logos

### Primary Wordmark
- **File:** `qx8-logo.svg` (light backgrounds)
- **File:** `qx8-logo-dark.svg` (dark backgrounds)
- **Usage:** Primary branding, website headers, presentations
- **Minimum width:** 180px
- **Clear space:** 20px on all sides

### Monogram/Mark
- **File:** `qx8-mark.svg` (light backgrounds)
- **File:** `qx8-mark-dark.svg` (dark backgrounds)
- **Usage:** App icons, social media avatars, small-space applications
- **Minimum size:** 48px × 48px

## Color Palette

All colors are HSL values for maximum design system compatibility.

### Primary Colors
- **Deep Navy** `hsl(224, 80%, 24%)` - `#0B1020`
  - Primary brand color, trust and stability
  - Use for headlines, primary CTAs, logo
  
- **Electric Blue** `hsl(217, 100%, 50%)` - `#2D6BFF`
  - Primary accent, innovation and energy
  - Use for interactive elements, CTAs, highlights

- **Teal Accent** `hsl(177, 100%, 38%)` - `#00C2B8`
  - Secondary accent, growth and success
  - Use for secondary CTAs, success states, accents

### Neutral Colors
- **Neutral Gray** `hsl(220, 25%, 96%)` - `#E9EDF2`
  - Background, cards, subtle surfaces
  
- **Onyx** `hsl(220, 90%, 5%)` - `#0A0A0A`
  - Body text, dark UI elements

## Typography

### Display Font (Headlines)
- **Family:** Inter (variable)
- **Weights:** 700 (Bold), 800 (Extra Bold)
- **Use:** H1, H2, H3, hero text
- **Letter spacing:** -0.02em

### Body Font
- **Family:** Source Sans 3
- **Weights:** 400 (Regular), 600 (Semi Bold), 700 (Bold)
- **Use:** Paragraph text, UI elements, forms
- **Line height:** 1.6

## Design Motifs

### Acceleration
The "X" in the logo features an arrow motif suggesting forward motion and velocity.

### Quantum
The superscript "8" represents discrete, quantized innovation steps - from pilot to deployment.

### Vector Lines
Use diagonal lines and converging arrows to suggest momentum and direction.

## Logo Usage Guidelines

### ✅ Do
- Maintain clear space around logos
- Use provided color variants for light/dark backgrounds
- Scale proportionally
- Use on backgrounds with sufficient contrast
- Pair with brand colors

### ❌ Don't
- Stretch or distort the logo
- Change logo colors (use provided variants)
- Add effects (shadows, outlines, gradients)
- Use on busy backgrounds without container
- Place logo smaller than minimum size

## File Formats

- **SVG:** Vector graphics, infinite scaling (preferred for web)
- **PNG:** Raster exports available at 16px, 32px, 48px, 180px, 512px
